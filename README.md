# jenn_freecam

Cinematic freecam for RedM with filters and cinematic bars.

## Dependencies

- [ox_lib](https://github.com/overextended/ox_lib)

## Features

- Free Camera Movement
- Camera Filters
- Cinematic Bars
- Zoom Control
- Max Distance limit from player

## Commands

| Command | Description |
|---------|-------------|
| `/freecam` | Open cinematic camera menu |
